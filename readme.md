# Windows Community Toolkit
Clone of [The Windows Community Toolkit](https://github.com/windows-toolkit) collection of helper functions, custom controls, and app services.  
It simplifies and demonstrates common developer patterns when building experiences for Windows 10.

## [Getting Started](https://docs.microsoft.com/windows/communitytoolkit/getting-started)

## [Documentation](https://docs.microsoft.com/windows/communitytoolkit/).
API documentation can be found at the [.NET API Browser](https://docs.microsoft.com/en-us/dotnet/api/?term=microsoft.toolkit).
